MTG Organizer

September 2020
Magic The Gathering (MTG) Organizer is a web application completely designed and developed by me with the occasional help from an instructor. It utilizes an API that allows for text input to search and return several objects that represent cards in game. It then copies the object into my own json server. React then rerenders the DOM to display an image taken from the returned object and an add to collection button. Once the user finds the current card, they can click the add to collection button to create a link between the user id and the card id. Under the users collection tab, the user can click an image of a card to display additional information, sort the cards based on color, and delete the card from their collection. Tools used: HTML, CSS, React, gitHub
