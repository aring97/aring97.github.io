Projects

Holiday Road

Aug 2020
Holiday road is an itinerary built using the National Park Services api and a weather service api to get up to date weather information for a selected park. It also included two api provided by NSS, one with a list of restaurants and the other with odd attractions. The user would be able to select a park, an attraction, and a restaurant that would then be saved to the groups own api. Based on the groups chosen persona we randomly recommended locations that included tags relevant to that persona. My role in the group was to retrieve and display weather information, render the save itinerary button after all three options were selected and provide the save itinerary button its functionality. tools used: HTML, CSS, JavaScript, postman, gitHub

MTG Organizer

September 2020
Magic The Gathering (MTG) Organizer is a web application completely designed and developed by me with the occasional help from an instructor. It utilizes an API that allows for text input to search and return several objects that represent cards in game. It then copies the object into my own json server. React then rerenders the DOM to display an image taken from the returned object and an add to collection button. Once the user finds the current card, they can click the add to collection button to create a link between the user id and the card id. Under the users collection tab, the user can click an image of a card to display additional information, sort the cards based on color, and delete the card from their collection. Tools used: HTML, CSS, React, gitHub
